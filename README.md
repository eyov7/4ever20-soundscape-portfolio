## Getting Started with This Project

This project is built with:
- Vite
- React
- shadcn-ui
- Tailwind CSS

### Local Development
```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm i

# Start development server
npm run dev
```
